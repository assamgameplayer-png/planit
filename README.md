# PLANIT 🪐

A playful and modern event scheduling web app built with pure HTML, CSS, and JavaScript.

## Features

- Monthly calendar view
- Event creation and editing
- Color categories
- Local storage persistence
- Mobile responsive UI
- Popup day planner

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
